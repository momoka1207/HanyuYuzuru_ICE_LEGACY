# Yuzuru Hanyu Dashboard

这是一个羽生结弦主题的静态 HTML Dashboard / Digital Exhibition。

## 如何发布到 GitHub Pages

1. 在 GitHub 新建仓库，例如 `yuzuru-hanyu-dashboard`。
2. 上传本文件夹里的所有文件，尤其是 `index.html`。
3. 进入仓库 `Settings` → `Pages`。
4. `Source` 选择 `Deploy from a branch`。
5. `Branch` 选择 `main`，目录选择 `/root`。
6. 等待 GitHub Pages 生成公开链接。

发布后链接通常是：

`https://你的GitHub用户名.github.io/yuzuru-hanyu-dashboard/`

## 后续如何更新

以后只要替换仓库里的 `index.html`，GitHub Pages 会自动更新网站。通常等待几十秒到几分钟即可生效。

## 说明

本项目为静态网页，不需要服务器、Python 或数据库。
